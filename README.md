# 🍽️ Dreams POS System

Dreams POS is a modern Point of Sale system built to simulate real restaurant operations with a strong focus on usability, interactivity, and workflow efficiency.

---

## 📌 Overview

This project delivers a complete POS experience across different devices.  
🖥️ Desktop is optimized for restaurant management, while 📱 mobile provides a simplified interface for end users.

All system pages are connected to reflect real-time changes across the application.

---

## 🚀 Key Features

### 🧾 Order Management
- Create and manage orders   
- Order statuses: Ordered, Preparing, Delivered  
- Seamless flow between POS, Orders, and Kitchen pages  

### 🍔 Dynamic Menu
- Categories and filters  
- Search functionality  
- Add-ons, sizes, and notes support  

### 🛒 Smart Cart System
- Real-time updates  
- Quantity control  
- Automatic price calculation  

### 💳 Payment System
- Multiple methods (Cash, Card, Wallet)  
- Change calculation  
- Receipt generation  

### 🔔 Notifications System
- New orders alerts  
- Interactive notification handling  
- Real-time updates  

### 📊 Reports & Insights
- Displays totals and percentages  
- Helps track performance and activity  
- Supports better decision-making  

### 📱 Responsive Experience
- 🖥️ Desktop → full restaurant control panel  
- 📱 Mobile → simplified user-friendly interface  
- Optimized for different screen sizes  

### 🎨 UI & UX
- Clean and modern design  
- Instant feedback on actions  
- 🌙 Dark mode support  

---

## 🧠 Tech Stack

- 🧱 HTML5  
- 🎨 CSS3  
- ⚙️ Vanilla JavaScript  

---

## 🔄 System Flow

1. 👨‍🍳 User places an order from POS  
2. 📦 Order is saved and reflected across all pages  
3. 🍳 Kitchen receives and updates order status  
4. 📊 Reports track activity and performance  
5. 🔔 Notifications inform about new updates
